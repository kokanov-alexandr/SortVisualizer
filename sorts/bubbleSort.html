function bubbleSort(arr) {
    for(var i = 0; i < arr.length; i++) {
        for(var j = 0; j < ( arr.length - i -1 ); j++) {

def bubbleSort(arr):
    n = len(arr)
    for i in range(n-1):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
                
                
            
                    
void swap(int *xp, int *yp) {
    int temp = *xp;
    *xp = *yp;
    *yp = temp;
}


void bubbleSort(int arr[], int n) {
    int i, j;
    for (i = 0; i < n-1; i++)
        for (j = 0; j < n-i-1; j++)
            if (arr[j] > arr[j+1])
                swap(&arr[j], &arr[j+1]);
}

Общее описание

Сортировка пузырьком, или Bubble Sort, — это простой алгоритм сортировки, который многократно проходит по списку, сравнивает соседние элементы и меняет их местами, если они находятся в неправильном порядке. Этот процесс повторяется фиксированное количество раз, равное количеству элементов в массиве.

Основные шаги алгоритма

  • Проход по массиву от начала до конца.
  • Сравнение каждого элемента с его следующим соседом.
  • Если элементы находятся в неправильном порядке (например, в случае сортировки по возрастанию первый элемент больше следующего), они меняются местами.
  • Повторение процессов сравнения и обмена n раз, где n — количество элементов в массиве.

Сложность алгоритма

Сценарий Сложность
Средний и худший случай O(n²)
Лучший случай (уже отсортированный массив) O(n)

Сортировка пузырьком является одним из самых простых алгоритмов сортировки. В своей базовой форме алгоритм неэффективен для больших списков, так как его время выполнения квадратично по числу элементов. Тем не менее, он часто используется в учебных целях из-за своей наглядности и простоты.
